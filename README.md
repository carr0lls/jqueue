# Job Queue

A simple RESTful API for retrieving and storing data from a specified URL.

## To run

```sh
npm install

// Transpile server-side files from ES6 to ES5
npm run build

// Start server
npm start
```

And visit <http://localhost:4000/>.

### Tests
```sh
// Run tests
npm run tests
```
